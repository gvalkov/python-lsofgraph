## Summary

Python port of [zevv/lsofgraph](https://github.com/zevv/lsofgraph).


## Usage

```bash
lsofgraph.py [opts] | dot -Tpng -o lsofgraph.png
lsof -n -F [opts] | lsofgraph.py | dot -Tpng -o lsofgraph.png
```
